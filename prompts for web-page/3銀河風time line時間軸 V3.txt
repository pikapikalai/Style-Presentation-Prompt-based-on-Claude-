# 銀河時間軸圖像記錄 (Graphic Recording) HTML 創建提示詞 V3

## 目的
請將以下內容轉換為以宇宙為主題、充滿夢幻且活潑氛围的時間軸形式圖像記錄HTML。在資訊設計和視覺設計兩個層面都追求最高水準。運用手繪風格元素與宇宙背景，創造神秘且視覺上極具吸引力的時間軸。

### 2. 宇宙主題的視覺效果
- 深色宇宙背景（#080808）散佈小星星
- 左上角配置大型「太陽」般的光源（使用調色盤主要色彩）
- 透過動畫讓光源產生呼吸般的動態效果
- 圓形標記設計為色彩繽紛的「行星」風格
- 整體應用星塵般的細緻背景紋理

### 3. 時間軸結構
- 中央設置垂直時間軸
- 項目左右交替配置
- 每個項目添加行星風格的圓形標記
- 年代顯示設計為浮現的卡片樣式
- 底部或右側配置快速導航選擇器
- 標記和選擇器添加發光效果

### 4. 字體排版與文字處理
- 標題：添加如夜空中閃耀星星般的發光效果
- 日期：以白色背景浮現的形式強調
- 事件標題：使用調色盤的明亮色彩顯示
- 詳細文字：半透明背景配明亮文字
- 字體指定：

### 5. 視覺元素與強調
- 使用 Font Awesome 圖示視覺化各事件內容
- 關鍵字突顯（漸層底線效果）
- 項目間保持適當留白確保可讀性
- 運用半透明效果和模糊營造景深感
- 滑鼠懸停時的動畫讓項目浮起

## 響應式設計對應
- 桌面版：左右分開的時間軸，底部配置選擇器
- 行動裝置直向：右側配置選擇器，時間軸靠中央
- 一般行動裝置：時間軸改為垂直單列，調整標記位置

## 特殊效果與動畫
- 太陽呼吸效果：`@keyframes glow { 0% { opacity: 0.7; } 100% { opacity: 1; } }`
- 行星標記光輝：`box-shadow: 0 0 5px 2px rgba(91,166,123,0.7);`
- 文字卡片懸停效果：`transform: translateY(-5px); box-shadow: 0 10px 20px rgba(0,0,0,0.15);`
- 導航項目懸停效果：讓說明向上浮現
- 突顯效果：`background: linear-gradient(transparent 60%, rgba(242,187,19,0.4) 60%);`

## 實作時注意事項
1. 一致使用色彩調色盤，維持符合宇宙主題的色調
2. 調整光源與星星配置，注意不要影響時間軸的可讀性
3. 調整時間軸項目間距，提供流暢的捲動體驗
4. 根據各事件性質適當選擇 Font Awesome 圖示
5. 響應式顯示時也要維持視覺衝擊力

## 整體指導方針
- 視覺化結合宇宙的無限延伸與時間流動
- 讓閱讀者能自然移動視線的配置與適當留白
- 視覺化明確呈現資訊的層級與關聯性
- 融合手繪風格與宇宙主題的獨特美感
- 用色彩區分各區段，視覺化呈現時代變化
- 一致使用字體、顏色、圖示營造統一感

## 具體範例：最適合年表、歷史事件、企業發展史、專案進度等
- 可依主題調整太陽顏色與背景氛圍
- 用不同顏色的標記區分各時代或階段
- 變更圖示直覺傳達各事件性質
